# Vertical Wordfill

現代詩向けの、輪郭語を本文の断片で埋める静的HTMLツールです。

公開URL:

https://t-takata-ainamic.github.io/vertical-wordfill/

## 使い方

`index.html` をブラウザで開きます。

- 輪郭語: 大きな形として描画する文字
- 埋める本文: 輪郭の内側にランダム/反復で配置するテキスト
- 生成モード: 文字単位、または単語比率
- 単語と割合: `単語 割合` を1行ずつ書くと、単語を途中で分割せずに指定比率へ近づけて配置
- 幅/高さ: 出力するテキストグリッド
- 密度: 輪郭内をどれくらい埋めるか
- seed: 同じ値なら同じ出力を再生成

出力はテキストエリアからコピーするか、`TXT保存` で保存できます。

## Notes

- ブラウザ内のCanvasで輪郭語をマスク化します。
- 外側と非充填部分は全角スペースです。
- 単語比率モードの出現数は、密度から必要な単語数を見積もり、割合に応じて四捨五入相当で配分します。
- 依存パッケージやビルド手順はありません。
